# sungseok_daughter 上传说明

把这个 `GITHUB` 文件夹里的内容全部上传到新仓库 `ChenFanqie37/sungseok_daughter` 的根目录。

## 必须上传

```text
index.html
app.js
styles.css
README.md
package.json
server.js
apikey.example.txt
.env.example
.gitignore
.nojekyll
.github/
scripts/
```

## 不要上传

```text
apikey.txt
api.txt
.env
*.docx
*.doc
server-job.id
server.pid
server.log
server.err.log
server-launcher.pid
```

## 开 GitHub Pages

1. 进仓库 `Settings`。
2. 点左侧 `Pages`。
3. 在 `Build and deployment` 里把 `Source` 选成 `GitHub Actions`。
4. 回到 `Actions`。
5. 运行 `Deploy Static Simulator`。

成功后的网页链接通常是：

```text
https://chenfanqie37.github.io/sungseok_daughter/
```

## API Key

GitHub 上不要放真实 key。真实 key 只放你电脑本地的 `apikey.txt`。

如果只打开 GitHub Pages 网页版，KKT 会使用本地兜底回复；如果要真实 LLM，需要在你自己的电脑或服务器上运行 Node 代理。
